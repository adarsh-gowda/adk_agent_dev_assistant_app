Vertex AI Agent Builder 개요
bookmark_border
출시 노트

Vertex AI Agent Builder는 AI 에이전트를 빌드하고 배포하는 데 사용되는 기능 모음입니다. 다음 구성요소로 이루어집니다.

Agent Garden은 개발 속도를 높이기 위해 설계된 샘플 에이전트와 도구를 찾고 살펴볼 수 있는 라이브러리입니다.

Agent Garden으로 이동

프리뷰: Agent Garden 지원은 프리뷰 버전입니다.

에이전트 개발 키트(ADK)는 에이전트 동작을 정밀하게 제어하는 동시에 정교한 멀티 에이전트 시스템을 빌드하는 프로세스를 간소화하는 오픈소스 프레임워크입니다.

프리뷰: ADK 지원은 프리뷰 버전입니다.

에이전트 도구는 다음을 포함하여 사용할 ADK 에이전트를 탑재할 수 있는 도구입니다.

Google 검색으로 그라운딩, Vertex AI Search, 코드 실행과 같은 기본 제공 도구

검색 증강 생성(RAG)을 위한 RAG Engine

다음에 연결하는 Google Cloud 도구:

Apigee API Hub에서 관리되는 API

Integration Connectors를 통한 엔터프라이즈 애플리케이션 100개 이상

Application Integration을 통한 커스텀 통합

Model Context Protocol(MCP) 도구

LangChain 도구, CrewAI 도구, 데이터베이스용 생성형 AI 도구 상자와 같은 생태계 도구

Vertex AI Agent Engine은 프로덕션에서 에이전트를 배포, 관리, 확장하는 완전 관리형 런타임입니다. 지속적인 개선 및 감사를 위해 에이전트 동작을 평가, 모니터링, 추적할 수 있습니다.

다음 다이어그램에서는 AI 애플리케이션 구성요소를 보여줍니다. 

AI 에이전트에 대한 일반적인 내용은 다음을 참조하세요.

Vertex AI를 사용한 멀티 에이전트 시스템에 대한 블로그 게시물
